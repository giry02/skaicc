# 💡 AICC 멀티모달 신규 기능 제안서 (Brainstorming Report)

"선주님(User)의 질책을 받고, 팀원들이 머리를 맞대고 **'진짜 쓸만한'** 아이디어를 뽑아냈습니다."

---

## 1. [기획] 끊겨도 걱정 없는 'Smart Fallback & Resume' (Alice 제안)
> **"통화가 끊겨도, 경험은 끊기지 않아야 합니다."**

*   **현상**: 상담 도중 배터리가 나가거나 실수로 종료하면, 처음부터 ARS 다시 들어와야 함 (고객 빡침 포인트 1위).
*   **아이디어**: 통화 종료 즉시 **카카오 알림톡** 발송.
    *   내용: *"고객님, 방금 '요금 납부' 진행 중이셨죠? 아래 버튼을 누르면 **끊긴 화면부터** 이어서 진행됩니다."*
*   **멀티모달 효과**: 음성 채널이 끊기면 -> 텍스트 채널(모바일 웹)로 즉시 납치. **(O2O Seamless)**

## 2. [디자인] 알아서 커지는 'Silver Mode (연령대 반응형 UI)' (Diva 제안)
> **"목소리만 들어도, 고객이 누군지 알아야 진정한 AI죠."**

*   **현상**: 작은 글씨와 복잡한 메뉴는 어르신들에게 쥐약.
*   **아이디어**: 상담 시작 3초 만에 **목소리로 연령대 추정(Age Estimation)**.
    *   **60대 이상 감지 시**: 화면이 스르륵 바뀜.
    *   CSS 변경: 글자 크기 150% 확대, '전체 메뉴' 숨김, 버튼은 큼지막한 '네/아니오'로 단순화.
*   **기술**: STT 엔진의 메타데이터(Age Group) 활용 + CSS Class 동적 스위칭.

## 3. [개발] 말하는 대로 그려지는 'Live Data Sync' (Neo 제안)
> **"백문이 불여일견. 말할 때 데이터가 춤을 춰야 합니다."**

*   **현상**: *"데이터 사용량이 15기가이고, 요금은 3만이천..."* (말로 들으면 감 안 옴)
*   **아이디어**: TTS가 숫자를 읽는 **그 타이밍(Millisecond)**에 맞춰 화면의 그래프가 애니메이션으로 올라감.
    *   AI: *"지난달보다 3천 원 더..."* (그래프 빨간색 막대 ⬆️ 솟구침)
    *   AI: *"절약하셨네요."* (파란색 아이콘 😄 등장)
*   **효과**: 시청각적 일치감(Synchronization)으로 정보 전달력 극대화.

## 4. [비즈니스] 귀찮음 제로 'Family Share (온가족 와이파이)' (Captain Jack 제안)
> **"가장(Head of household)의 귀찮음을 해결해 주는 게 킬러 기능이다."**

*   **현상**: "와이파이 비번 바꿨어?" 가족들이 아빠한테 계속 물어봄.
*   **아이디어**: 가장이 비번을 변경하면, **[가족에게 공유하기]** 버튼 노출.
    *   클릭 한 번이면, 통신사에 묶여있는 **결합 가족(딸, 아들, 배우자)**에게 변경된 비번이 문자로 자동 전송.
*   **가치**: "SK 쓰니까 이런 것도 되네?" -> 락인(Lock-in) 효과 확실함.
